# Instagram Data Analysis

## Objective

The objective of this project is to analyze Instagram user activity, engagement patterns, hashtags, followers, likes, comments, and content performance using Exploratory Data Analysis (EDA). The analysis helps identify trends and provide recommendations for improving social media engagement.

## Dataset

The dataset contains information related to:

* Users
* Photos
* Likes
* Comments
* Followers
* Hashtags
* Photo-Hashtag Relationships

### Dataset Files

* users.csv
* photos.csv
* likes.csv
* comments.csv
* follows.csv
* tags.csv
* photo_tags.csv

## Analysis Performed

* Dataset Overview
* Missing Values Analysis
* Data Types Analysis
* Hashtag Analysis
* User Activity Analysis
* Most Liked Posts Analysis
* Follower Analysis
* Account Verification Analysis
* Public vs Private Account Analysis
* Instagram Filter Analysis

## Key Findings

* The hashtag "smile" was the most frequently used hashtag.
* User 23 was the most active user in the dataset.
* Photo ID 145 received the highest number of likes.
* Most accounts were non-verified.
* Public accounts were more common than private accounts.

## Recommendations

* Use trending hashtags consistently.
* Post educational and engaging content.
* Encourage audience interaction.
* Maintain a consistent posting schedule.
* Monitor engagement metrics regularly.
